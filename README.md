# Haideri Medical and Dental Center

A modern, professional website for Haideri Medical and Dental Center in Abbottabad.

## Overview

This project presents the clinic as a trusted, patient-focused medical and dental provider with:

- a polished homepage for first impressions
- service pages for dental and medical care
- a contact page with phone, WhatsApp, and location details
- responsive design for desktop and mobile users

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- lucide-react

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser at:

```text
http://localhost:3000
```

## Build

```bash
npm run build
```

## Project Structure

- src/app - pages and routes
- src/components - reusable UI components
- src/lib - clinic data and shared content
